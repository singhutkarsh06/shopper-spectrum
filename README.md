# 🛒 Shopper Spectrum
### Customer Segmentation & Product Recommendation System

## 📌 Project Overview
Shopper Spectrum is a data science project that analyzes 
e-commerce transaction data to segment customers and 
recommend products using machine learning techniques.

## 🎯 Key Features
- Customer Segmentation using RFM Analysis & KMeans Clustering
- Product Recommendation using Collaborative Filtering
- Interactive Streamlit Web Application
- Exploratory Data Analysis Dashboard
- Real-time Predictions

## 🧠 Machine Learning Techniques
- RFM Analysis (Recency, Frequency, Monetary)
- KMeans Clustering
- Cosine Similarity based Collaborative Filtering
- StandardScaler Normalization

## 📊 Customer Segments
| Segment | Description |
|---|---|
| 💰 High-Value | Frequent, recent, big spenders |
| ✅ Regular | Steady purchasers |
| 🔵 Occasional | Rare purchases |
| ⚠️ At-Risk | Haven't purchased in a long time |

## 🛠️ Technologies Used
- Python
- Pandas & NumPy
- Scikit-learn
- Matplotlib & Seaborn
- Streamlit
